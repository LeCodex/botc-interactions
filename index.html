Unofficial BotC Script Interactions Tool

This website is a tool to do an automated check of your BotC script (using the JSON files created by the Script Tool). This will allow you to detect interactions between certain characters, in order to think about if those fit the script you are building.

Messages given by this tool are not perfect nor are they a reason to rework your script: they are voluntarily very exhaustive, and are just interactions you might need to think about in order to smooth out the edges of your script, but it isn't required for every one of them to be fully resolved.

Blood on the Clocktower is a game by Steven Medway, published by The Pandemonium Institute. More information at bloodontheclocktower.com.

All interactions are written in this document. If you wish to suggest changes, please leave a comment on it.
